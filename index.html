Calcular el área de un rectángulo

  1. Iniciar.
  2. Leer la base del rectángulo.
  3. Leer la altura del rectángulo.
  4. Calcular el área = base × altura.
  5. Mostrar el área.
  6. Terminar.

Calcular el promedio de N notas y mostrar si el estudiante aprueba o desaprueba.

  1. Iniciar.
  2. Leer la cantidad de notas (N).
  3. Inicializar suma = 0.
  4. Repetir desde i = 1 hasta N:
    1. Leer la nota i.
    2. Acumular suma = suma + nota.
  5. Calcular promedio = suma / N.
  6. Si promedio ≥ 11, entonces:
    1. Mostrar "Aprobado".
    2. Si no, mostrar "Desaprobado".
  7. Terminar.
